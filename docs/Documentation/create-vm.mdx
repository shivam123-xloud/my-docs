# Create a Virtual Machine

<Image
  src="/images/creating-vm/1.png"
  alt="Xloud portal login page"
/>

A **Virtual Machine (VM)** — also known as an **Instance** — is a virtualized compute resource running on cloud infrastructure. It allows you to deploy and run applications without managing physical hardware.

---

## Before You Begin

<Card title="Key Concepts" icon="cloud">
  <ul>
    <li>
      <strong>Subnet</strong>
      <br />
      A logical division of an IP network used for segmentation and IP address management.
    </li>
    <li>
      <strong>ILL (Internet Leased Line)</strong>
      <br />
      A dedicated, high-speed connection for external or internet connectivity.
    </li>
  </ul>
</Card>

<Callout type="note">
  If you don’t have a subnet yet, refer to <strong>How to Create a Subnet</strong> before continuing.
</Callout>

---

## Create a Virtual Machine

<Steps>

<Step title="Log in to the Xloud Portal">
  Open the Xloud portal and sign in using your credentials.

  <a href="https://portal-del.xloud.tech/auth/login/?next=/" target="_blank">
    Login to Xloud Portal
  </a>

  <Image
    src="/images/creating-vm/1.png"
    alt="Xloud portal login screen"
  />
</Step>

<Step title="Navigate to Instances">
  Follow the navigation path:
  <ul>
    <li><strong>Project</strong></li>
    <li><strong>Compute</strong></li>
    <li><strong>Instances</strong></li>
    <li>Click <strong>Launch Instance</strong></li>
  </ul>

  <Image
    src="/images/creating-vm/2.png"
    alt="Navigate to instances page"
  />

  <Image
    src="/images/creating-vm/3.png"
    alt="Launch instance button"
  />

  <Callout type="tip">
    All virtual machines are managed from the <strong>Instances</strong> page.
  </Callout>
</Step>

<Step title="Configure Instance Details">
  Provide basic information for your VM:
  <ul>
    <li><strong>Instance Name</strong> — Choose a meaningful name</li>
    <li><strong>Description</strong> — Optional</li>
    <li><strong>Count</strong> — Number of instances to create</li>
  </ul>

  <Image
    src="/images/creating-vm/4.png"
    alt="Instance details configuration"
  />

  <Callout type="warning">
    Ensure your project has sufficient quota before creating multiple instances.
  </Callout>
</Step>

<Step title="Select the Operating System Source">
  Configure the OS disk settings and select an image.

  <Image
    src="/images/creating-vm/5.png"
    alt="Select operating system image"
  />

  <Callout type="warning">
    Select <strong>No</strong> if you want to preserve data after deleting the instance.
  </Callout>
</Step>

<Step title="Choose a Flavor">
  Select compute resources such as CPU and RAM.

  <Image
    src="/images/creating-vm/6.png"
    alt="Choose instance flavor"
  />
</Step>

<Step title="Configure Networking">
  Attach the required networks:
  <ul>
    <li><strong>ILL</strong></li>
    <li><strong>Subnet</strong></li>
  </ul>

  <Image
    src="/images/creating-vm/7.png"
    alt="Configure instance networking"
  />

  <Callout type="tip">
    Selecting both <strong>ILL</strong> and <strong>Subnet</strong> helps optimize networking and reduce IP-related costs.
  </Callout>
</Step>

<Step title="Configure Security Groups">
  Manage firewall rules for your instance.

  <Image
    src="/images/creating-vm/8.png"
    alt="Security group selection"
  />

  <Callout type="note">
    Refer to <strong>How to Create Security Groups</strong> for custom firewall rules.
  </Callout>
</Step>

<Step title="Select a Key Pair">
  Choose a key pair for secure access.

  <Image
    src="/images/creating-vm/9.png"
    alt="Select key pair"
  />

  <Callout type="danger">
    Never share your private key. Store it securely — it cannot be recovered if lost.
  </Callout>
</Step>

<Step title="Launch the Instance">
  Click <strong>Launch Instance</strong> to create the VM.

  <Image
    src="/images/creating-vm/10.png"
    alt="Launch virtual machine"
  />

  After launch, view the VM at:
  <br />
  <strong>Project → Compute → Instances</strong>

  <Image
    src="/images/creating-vm/11.png"
    alt="Instances list page"
  />
</Step>

</Steps>
