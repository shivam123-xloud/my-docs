## Frequently Asked Questions

<Accordion title="Is there a separate license for the hypervisor?">
  No. The Xloud hypervisor is included in the platform license. No third-party hypervisor license (VMware, Hyper-V) is required.
</Accordion>

<Accordion title="Are there per-VM or per-socket fees?">
  No. Xloud uses a subscription model with no per-VM, per-socket, per-core, or per-host fees. Scale infrastructure without incremental license costs.
</Accordion>

<Accordion title="Are advanced features (Key Manager, DNS, Load Balancer) separately licensed?">
  No. Every Xloud service is included in the base license and enabled from Day 1. There are no add-on modules or premium tiers.
</Accordion>

<Accordion title="What happens when the subscription expires?">
  The software continues to operate. However, access to updates, security patches, and OEM support requires an active subscription.
</Accordion>

<Accordion title="Can the software be modified or customized?">
  Yes. The Apache 2.0 license permits modification, customization, and integration without obligation to open-source derivative works.
</Accordion>

<Accordion title="Are there restrictions on the number of users or projects?">
  No. User and project counts are limited only by administrator-configured quotas — not by licensing.
</Accordion>
